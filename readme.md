# 3DModelHub

A professional, SEO-optimized landing page for showcasing CGTrader 3D models.

## 🚀 Deployment on Netlify

1. **Prepare the files**: Ensure `index.html`, `styles.css`, and `script.js` are in the root directory.

2. **Zip the project** (optional): Create a zip file containing all files.

3. **Deploy to Netlify**:
   - Go to [Netlify](https://netlify.com)
   - Drag and drop the folder or zip file into the deployment area
   - Or connect your Git repository if using version control

4. **Configure domain** (optional): Set up a custom domain in Netlify settings.

## 📝 Customization

### Adding Models
1. Open `index.html`
2. Find the `<!-- Model Card Template -->` comment
3. Copy the `<div class="card">` block
4. Paste it inside the `.grid` container
5. Replace:
   - `Model Title Placeholder` with your model title
   - The `<p>` text with your model description
   - `href="#"` with your CGTrader model URL

### SEO Setup
- Update the Google verification meta tag with your actual code
- Modify meta description and keywords as needed

### Styling
- Edit `styles.css` to customize colors, fonts, or layout
